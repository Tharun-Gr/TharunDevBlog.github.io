Executive Summary

This centralised software system is used to streamline the controls of various devices such as projectors, audio and video controls etc.

My Role

In my role as a Lead Developer, I spearheaded the design and implementation of a centralized software system to manage a wide array of devices, including projectors and audio equipment, across five campus locations encompassing over 800 university spaces.

Technical Highlights

CI/CD Refinement

Revamped the CI/CD pipeline using Jenkins, SonarQube, and Docker for streamlined deployment processes.

Kubernetes Infrastructure

Enhanced infrastructure scalability by implementing Argo CD for Kubernetes deployment management.

Database Integration

Ensured consistent performance and state management with centralized database systems for device information and monitoring logs.

Architecture

Key Features

Device Management

Developed a system enabling real-time interaction with devices through a user-friendly interface.

Real-time Monitoring

Integrated web socket communication to allow real-time status updates and control of each device.

Automated Device Registry

Engineered an automatic device registration system to efficiently handle new installations and configurations.

Impact

The deployment of this system has significantly improved the operational efficiency of multimedia device management across the university, resulting in enhanced learning experiences and administrative productivity.

Conclusion

This project stands as a testament to my ability to lead complex software development initiatives and drive technological innovation in an educational environment. By leveraging modern development practices and fostering a collaborative team atmosphere, we delivered a system that not only meets the current needs but is also poised for future expansion and integration.
